Persistent Local Storage: Uses the chrome.storage.local API to save tracking data, ensuring that your time history remains available even after closing the browser.  Productivity Dashboard: A dedicated popup UI displays a list of visited sites along with the total seconds spent on each.  Smart Filtering: The tracking logic is designed to ignore invalid URLs, such as internal chrome:// pages, to keep analytics accurate.  Total Usage Summary: Calculates the aggregate time spent across all tracked websites for a comprehensive daily overview.
